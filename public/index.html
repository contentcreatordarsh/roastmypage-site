        LATEST Mar 10, 2026
        🔌
        Developer API v1

        Public API for developers. POST /api/v1/roast to analyze any landing page programmatically. GET /api/v1/usage to check your limits. 5 roasts/day per IP, 50/day globally. Open CORS, no API key required. Check the new API button in the nav bar.

        🔒
        Security Hardening

        Major security audit and 21 vulnerability fixes across two passes. SQL injection prevention, XSS sanitization on all dynamic content, CSRF protection, parameterized queries, SVG escaping, and rate limit race condition fix.

        🛡
        Hardened CSP & Headers

        Removed unsafe-eval from Content Security Policy, added security headers to all server-rendered pages (roast share pages, gallery, 404), environment-aware CORS origins, and generic error responses to prevent information leaks.

        🗂
        Codebase on GitLab

        Project source code is now version-controlled and pushed to GitLab. Proper README with project overview, tech stack, and deployment docs.

        🛠
        Infrastructure Cleanup

        Moved hardcoded account/zone IDs to environment variables, configurable IP hash salt, 5MB screenshot size limits, atomic rate limiting to prevent race conditions, and safe error logging with URL redaction.
